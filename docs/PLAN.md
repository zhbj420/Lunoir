# MMPlayer 方案与里程碑

> 一款以 mpv 为内核、界面复刻 macOS 上 IINA 播放器的 Windows 媒体播放器。

## 1. 目标

- **内核**：复用 mpv（libmpv/mpv.exe），不重复造解码/播放轮子。
- **外观**：参考 IINA —— 无边框窗口、半透明毛玻璃悬浮控制条、精致动画、自动隐藏 OSC。
- **路线**：先做最基础的播放功能，跑通后再逐步增加 IINA 的进阶功能。

## 2. 技术选型（已确认 2026-07-16）

| 决策 | 选择 | 理由 |
| --- | --- | --- |
| 应用框架 | **Electron + React + TypeScript**（electron-vite） | 已有 Node 环境；Web 技术还原 IINA 精致界面最快最像；迭代快 |
| 内核集成 | **自动下载内置 mpv.exe** | 用户开箱即用，无需自行安装 |
| mpv 下载源 | shinchiro/mpv-winbuild-cmake（GitHub Releases） | zhongfly 源已 404；shinchiro 稳定滚动更新 |
| 未采用 | Tauri（需装 Rust）、C++/Qt（界面成本高） | — |

## 3. 架构设计

```
┌─────────────────────────────────────────────┐
│  Electron 主进程 (src/main)                   │
│   · 双窗口管理 / 菜单 / 文件对话框              │
│   · 启动 mpv，取窗口句柄 HWND，经 --wid 嵌入   │
│   · 转发 mpv 属性事件到 UI                     │
├─────────────────────────────────────────────┤
│  mpv.exe (resources/mpv)                      │
│   · 视频渲染到「视频窗口」                      │
│   · JSON IPC 命名管道控制（播放/seek/音量…）   │
├─────────────────────────────────────────────┤
│  渲染进程 React (src/renderer)                │
│   · 透明「UI 窗口」浮在视频窗口之上             │
│   · IINA 风格悬浮 OSC / 标题栏 / 空状态         │
│   · 自行处理鼠标/键盘手势，经 IPC 下发          │
└─────────────────────────────────────────────┘
```

**双窗口方案**：一个无边框「视频窗口」承载 mpv 画面；一个透明「UI 窗口」（视频窗口的子窗口）浮在上层承载 React 控制界面。UI 窗口为用户实际拖动/缩放的窗口，视频窗口跟随其几何变化。这是在 Windows 上把自定义 HTML 界面叠加到 mpv 画面之上的标准做法。

**通信**：preload 通过 `contextBridge` 暴露 `window.mmp`；渲染进程发指令 → 主进程 → mpv IPC；mpv 属性变化 → 主进程 → 渲染进程。

## 4. 里程碑

| 阶段 | 内容 | 状态 |
| --- | --- | --- |
| **M0 脚手架** | electron-vite + React + TS；`npm run setup` 自动下载 mpv | ✅ 完成 |
| **M1 控制回路** | Node ↔ mpv IPC 打通；打开文件、播放/暂停/seek | ✅ 完成 |
| **M2 嵌入 + 界面** | mpv `--wid` 嵌入 + 透明覆盖层 OSC；进度/时间/音量/全屏/自动隐藏/拖拽/快捷键 | ✅ 完成（待用户视觉确认） |
| **M3 打磨 + 打包** | 依实际效果调界面；处理透明窗口缩放边缘情况；electron-builder 打包安装程序 | ⏳ 进行中 |

## 5. 第一期功能范围（MVP）

- 打开视频：拖拽 + 菜单/按钮 + Ctrl+O
- 播放 / 暂停、进度条拖动跳转、当前/总时长
- 音量调节 + 静音、滚轮调音量
- 全屏切换（F / 双击）
- IINA 风格：无边框、半透明圆角控制条、自动隐藏、空状态引导
- 快捷键：空格/K 播放暂停、←→ 快退快进 5s、↑↓ 音量、F 全屏、M 静音

## 6. 后续路线（第一期之后，按 IINA 功能）

播放列表 → 字幕/音轨切换 → 倍速播放 → 截图 → 画质滤镜/色彩调节 → 记忆播放进度 → 缩略图预览 → 多标签 …
